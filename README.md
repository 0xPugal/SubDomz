# subdomz
**A Complete Subdomain Enumeration Tool** 

The tool SubDomz uses various multiple open-source tools and OSINT search engines to gather subdomains.

## Usage
```
git clone https://github.com/0xlittleboy/subdomz
cd subdomz
sudo bash install.sh
bash subdomz.sh -h
```

## Tools Used
+ Subfinder
+ Findomain
+ Amass
+ Assetfinder
+ Sublist3r
+ Waybackurls
+ Gauplus
+ Chrobat
+ Censys-subdomain-finder
+ github-subdomains
+ CTFR
+ Shodomain

## Search Engines
+ Wayback Machine
+ BufferOver.run
+ Riddler.io
+ Certspotter
+ Archive
+ JLDC
+ Crt.sh
+ Security Trails
+ CommonCrawl
+ HackerTarget
+ ThreadCrowd
